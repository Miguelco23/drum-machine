# React Drum Machine 🥁

This is a proyect made for the *FreeCodeCamp* Frontend libraries course

I made it with [React.js](https://reactjs.org/) Html, and Css

## Setup 🤖
* Install [node.js](https://nodejs.org/es/)
* Finally just `npm start` and that's it!
